### Design Principles

Every F² product follows these rules:

1. **Solves a specific, measurable problem** — not "AI for everything"
2. **Architecturally complete** — not a demo, not a pitch deck
3. **Ships with explainability** — every AI decision can be traced and audited
4. **Integrates with existing systems** — ERP, accounting, IAM, compliance tooling
5. **Runs on production infrastructure** — GCP, Docker, Kubernetes, automated CI/CD
